You are a senior Python systems engineer.
Design and implement a **production-ready tracing system for Google ADK agents** using OpenTelemetry.

The goal is to allow developers to install tracing with a single line:

```python
import agent_tracing

agent_tracing.install_gcs_tracing(bucket="agent-traces")
```

After this call, all OpenTelemetry spans generated by ADK agents should automatically be captured and stored in Google Cloud Storage.

The system must be **non-blocking**, **scalable**, and **UI-queryable through a FastAPI service**.

---

1 convert OpenTelemetry spans → SpanRecord

3 never block
4 drop spans if queue is full

20000

BACKGROUND WRITER

Implement:

class GCSWriter

Runs in a daemon thread.

Responsibilities:

* consume spans from queue
* group spans by (session_id, query_id)
* buffer spans
* periodically flush to GCS

Flush conditions:

500 spans
OR
20 seconds

GCS STORAGE LAYOUT

Store traces using session-based partitioning:

gs://agent-traces/

```
session_id=<SESSION_ID>/

    session_index.json

    query_id=<QUERY_ID_1>/
        spans_<timestamp>.ndjson.gz

    query_id=<QUERY_ID_2>/
        spans_<timestamp>.ndjson.gz
```

SPAN FILE FORMAT

Use compressed NDJSON.

Example:

{

"span_id": "3",
"parent_span_id": "2",
"session_id": "s1",
"query_id": "q1",
"span_name": "tool.sql",
"span_type": "tool",
"duration_ms": 220
}

---

SESSION INDEX

Each session folder contains:

session_index.json

Purpose:

fast lookup of queries without scanning span files.

Example structure:

{
"session_id": "s1",
"created_at": 1710638200,

"queries": {
"q1": {
"trace_id": "abc",
"start_time": 1710638210,
"end_time": 1710638215,
"duration_ms": 530,
"span_count": 12,
"agents": ["router_agent"],
"tools": ["sql_tool"],
"files": [
"query_id=q1/spans_1710638210.ndjson.gz"
]
}
}
}

Writer must update this file when a query completes.

EXAMPLE SCRIPT

Provide example:

example.py

```
import agent_tracing
from opentelemetry import trace

agent_tracing.install_gcs_tracing(
    bucket="agent-traces"
)

tracer = trace.get_tracer(__name__)

with tracer.start_as_current_span("agent.router"):
    with tracer.start_as_current_span("tool.sql"):
        print("executing tool")
```
